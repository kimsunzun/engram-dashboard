# engram-dashboard

Engram — Claude 에이전트 관리 데스크톱 앱.

**설계·진행 문서: [`docs/README.md`](docs/README.md)** (전체 인덱스 — 구조·진행 상태·다음 작업)

## 기술 스택

- Tauri v2 (데스크톱 껍데기, Rust 백엔드 + portable-pty)
- React 19 + TypeScript + Vite (프론트엔드)
- xterm.js (터미널 렌더링)
- Monaco DiffEditor (코드 변경 diff 뷰)
