# Step 6 QA — 슬롯 컴포넌트 검증

## 할 일

1. `npm run tauri dev` 실행 (I:\Engram\apps\engram-dashboard\)
2. chrome-devtools MCP로 스크린샷 + 인터랙션 확인

## 체크리스트

| 항목 | 기대값 |
|------|--------|
| 포커스 테두리 | 클릭 슬롯 2px accent색, 나머지 1px border색 |
| 에이전트 오버레이 | 슬롯 우하단에 에이전트 이름 (미할당 시 "—") |
| 트리→슬롯 연동 | 트리 노드 클릭 → 포커스 슬롯 오버레이 이름 변경 |
| 우클릭 메뉴 | 분할/에이전트 전환/닫기 항목 표시 |
| 메뉴 닫힘 | 외부 클릭 시 컨텍스트 메뉴 사라짐 |
