# QA — 슬롯 동적 분할

## 할 일

1. `npm run tauri dev` 실행 (I:\Engram\apps\engram-dashboard\)
2. chrome-devtools MCP로 인터랙션 확인

## 체크리스트

| 항목 | 기대값 |
|------|--------|
| 초기 상태 | Slot1 / Slot2 수평 2분할 |
| 가로 분할 | 슬롯 우클릭 → "가로 분할" → 해당 슬롯이 좌우 2개로 분할 |
| 세로 분할 | 슬롯 우클릭 → "세로 분할" → 해당 슬롯이 상하 2개로 분할 |
| 중첩 분할 | 분할된 슬롯을 다시 분할 가능 |
| 닫기 | 슬롯 우클릭 → "닫기" → 해당 슬롯 제거, 나머지 재배치 |
| 마지막 슬롯 닫기 | 슬롯 1개 남았을 때 닫으면 새 슬롯 1개로 대체 (앱 깨지지 않음) |
| 포커스 유지 | 닫힌 슬롯이 포커스였으면 남은 슬롯 중 하나로 포커스 이동 |
