## 함정 (do-not)
- **defaults 이중 출처:** `CHAT_STYLE_DEFAULTS`(chatStyleStore.ts) ↔ `theme.css :root` 9값 — 값 바꿀 땐 **둘 다** 동기(drift 감지 테스트가 chatStyleStore.test.ts에서 지킴).
- **`window.__engramChat` 키 화이트리스트** = `CHAT_STYLE_KEYS` 고정 배열 멤버십. **`key in CHAT_STYLE_DEFAULTS`로 되돌리지 말 것** — `in`은 프로토타입 체인을 타서 `constructor`/`__proto__` 통과(prototype pollution, Codex가 적출).
- **StructuredTextView 순수 렌더 유지** — state/effect 추가 금지(ADR-0050). run-position은 렌더 중 파생 계산.
- **불변식(손대지 말 것):** dispatch FIX1(tool_result 흡수)/FIX2(InertCode)/FIX3(`__streaming__` key)·usage 무렌더·structuredAccumulator(ADR-0045).
- **opus thinking = 암호화(평문 없음)라 "Thought" 라벨만 뜨는 게 정상** — 재디버깅 금지(ADR-0049). sonnet은 평문.
- **BOM 회귀** — 프론트 파일 편집 후 첫 3바이트 EF BB BF 검사.
- **앱 재기동:** `run-dashboard-clean.bat`은 데몬 재빌드(`cargo build -p daemon`)까지 함 → 미커밋 Rust echo 컴파일(빌드 깨질 소지). 프론트만이면 데몬 재빌드 없이 `$env:WEBVIEW2_ADDITIONAL_BROWSER_ARGUMENTS="--remote-debugging-port=9223"; npm run tauri dev`. 포트 9223.
