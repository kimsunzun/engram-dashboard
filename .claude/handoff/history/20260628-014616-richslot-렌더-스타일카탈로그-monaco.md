## 정해진 것 (= ADR 대상, 아직 ADR 안 씀)
- **슬롯 렌더 = 대화형(chat).** thinking/도구 = **progressive disclosure**(활성=펼침 / 끝나면 옅은 한 줄 접힘 / 클릭 재펼침). 라이브 전이는 주석으로만 박음(스트림 아직).
- **색 = 보조 신호**(유일 신호 금지 — e-ink 모노에서도 구분 살아야). 표현 = **per-View 템플릿(preset)으로 통째 교체**, 사람 UI와 LLM이 같은 핸들(§5).
- **코드/diff = Monaco(무의존, VS Code 그대로) 우선.** 인라인 경량 = **Shiki는 미정**(새 의존성 → dashboard2 package.json 조율 + ADR).
- **마크다운 = 자체 미니(무의존) 채택**, streamdown은 미정.

## 실패/주의 (do-not)
- claude `-p` 실측 캡처 `--permission-mode bypassPermissions` → **안전 분류기 차단**. 그래서 fixture는 **합성**(showcase-gen.mjs 등). 실측 캡처 필요하면 사용자 인가 받아야.
- **diff monaco를 일반 `<Editor language="diff">`로 하면 색 안 나옴**(vs-dark diff 토큰색 거의 없음). → **DiffEditor(renderSideBySide:false) 인라인**이라야 초록/빨강. (MonacoCode.tsx `MonacoDiff` 참고.)
- **새 npm 의존성 추가 금지**(package.json = dashboard2 공유). Monaco는 *기존* 의존성이라 OK.
