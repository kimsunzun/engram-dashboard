# 핸드오프: S15 모듈2(B2 decoder) 완료·커밋 + 모듈3(B4+B5 Ring/payload-generic) 완료·게이트 PASS·**미커밋**, 다음=모듈4 B7+B8

## ⭐ 이 세션에서 바뀐 규약 — 커밋 금지 (사용자 결정) ⭐
사용자가 **"자율 모드에서도 자동 커밋 금지"**를 지시했다. 앞으로 **커밋은 사용자가 명시적으로 지시할 때만** 한다 — "진행 쭉해"·"계속 진행"은 코더→리뷰→qa 파이프라인 진행 지시이지 커밋 승인이 아니다. 게이트가 다 통과해도 커밋 메시지 초안 + 스테이징 대상만 제시하고 사용자 승인을 기다린다.
- 이 규약을 `implement` 스킬에 반영 완료(SKILL.md + flow.md §5 — 미커밋, `/review doc` 후 커밋 대상). **로드된 스킬은 hot-reload 안 되므로 다음 세션부터 자동 적용.**
- 사용자 불만 맥락: 자동 커밋이 워킹트리를 clean으로 만들어 **커밋 전 diff 검토 기회를 뺏는다**. 커밋 전 반드시 보여주고 승인받을 것.

## ⭐ 리뷰가 잡은 버그 / do-not (재도입 금지) ⭐
적대 리뷰가 코드로 적출·교정한 것들. 다시 만들지 말 것:
1. **Ring이 최신 이벤트를 통째 evict(Codex High).** cost_bytes > max_bytes(2MB)인 단일 이벤트면 eviction 루프가 방금 넣은 최신 seq까지 빼내 replay가 빈다 → 늦은 구독자가 최신 놓침. **교정: `while items.len() > 1 && (over budget)`** — 최신 1건 항상 보존. (구조화 이벤트 큰 args_json이 B3 배선 시 발현하는 시한폭탄이었음.)
2. **result.is_error 은닉 + interrupt 오분류(B2).** result 라인 오류 판정은 **error allowlist**(`subtype.starts_with("error")` 또는 `is_error==true`)여야 함. `!= "success"` denylist는 정상 `interrupted`(유저 Esc)를 실패로 위장 → 금지.
3. **malformed 블록이 가짜 정형 이벤트(B2).** name 없는 tool_use는 빈 ToolCall 대신 `Structured` 보존, text 없으면 skip.
4. **오버플로 후 resync 부재(B2).** 오염 라인 clear만 하면 꼬리가 가짜 이벤트 유발 → `discarding` 상태로 다음 개행까지 폐기.
5. **snapshot() 구조화 무음 유실(모듈3, 3렌즈 수렴).** `snapshot()->Vec<OutputChunk>`가 구조화 이벤트를 drop(TerminalBytes만 wire). 현재 `_ => None`에 `tracing::warn!` 관측 훅 부착됨. **B7이 구조화→wire 매핑을 실제로 해야** 이 유실이 닫힘 — 모듈4에서 처리. 그전에 B3(생산자) 배선 시 무음 유실 주의.
6. **fixtures git 스테이징 필수(B2, Codex).** `include_str!` 의존 파일은 커밋에 함께(B2 때 반영됨).

## 확정 설계 (정본 = ADR-0045 + S15 TRD)
- 데이터 흐름: claude NDJSON → StdioTransport pump(바이트) → **decoder(B2✅, backend 소유)** → OutputEvent → **OutputCore Ring<StoredOutput>(B4✅, ADR-0006 락 규율)** → **OutputPayload{Bytes|Event}(B5✅)** → **daemon adapter(B7, core→wire)** → WsOutputSink(tag0 bytes/**tag1 event=B7**) → codec decode_frame(헤더-only) → 클라 공유버퍼 append/replay 재사용 → 프론트 wsFrame tag 분기(F).
- 불변식: ADR-0006(emit lock 밖 send·replay+subscribers 동시 보유 금지, subscribe만 예외 C4) · ADR-0003(core는 wire 모름, Serialize 미부착, tauri import 0) · ADR-0004(claude 스키마 backend/claude.rs만) · ADR-0002(출력 종류 비가정 = Bytes/Event 분기).
- cost_bytes = eviction 예산용 **근사**(core 직렬화 불가라 payload String 길이 합). 정확한 wire 크기 아님.
- WsOutputSink Event arm = **dormant guard**(debug_assert!+warn+Ok) — B7이 tag1 실인코딩으로 대체. Err 반환 금지(멀쩡한 sink 오제거).
